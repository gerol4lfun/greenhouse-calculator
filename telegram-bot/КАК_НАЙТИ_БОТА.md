# 🔍 Как найти Telegram бота в Railway

## ❓ Какой проект содержит бота?

В Railway у вас несколько проектов. Нужно найти тот, где есть сервис с Telegram ботом.

---

## 🎯 Способ 1: Проверить каждый проект

### Шаг 1: Кликните на проект

Кликните на каждый проект по очереди:
- `calm-exploration` (зеленый, online)
- `protective-nourishment` (зеленый, online)
- `remarkable-truth` (не online)
- `airy-respect` (красный, crashed)
- `jubilant-youth` (красный, crashed)

### Шаг 2: Проверьте список сервисов

После клика на проект вы увидите список сервисов внутри проекта.

**Ищите сервис с названием:**
- `telegram-bot`
- `delivery-dates-bot`
- `bot`
- Или что-то связанное с "delivery" или "dates"

### Шаг 3: Проверьте настройки сервиса

Если нашли подходящий сервис:
1. Кликните на него
2. Перейдите в **"Settings"**
3. Проверьте раздел **"Source"**
4. Должен быть подключен ваш GitHub репозиторий (например, `greenhouse-calculator`)

---

## 🎯 Способ 2: По названию репозитория

Если вы помните название репозитория на GitHub:

1. Кликните на проект
2. Найдите сервис с иконкой GitHub (Octocat)
3. Под иконкой должно быть название репозитория
4. Если это ваш репозиторий с калькулятором - это правильный проект

---

## 🎯 Способ 3: По переменным окружения

Если нашли сервис, который похож на бота:

1. Кликните на сервис
2. Перейдите в **"Settings"**
3. Найдите раздел **"Variables"** (Переменные окружения)
4. Должны быть переменные:
   - `TELEGRAM_BOT_TOKEN`
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_ROLE_KEY`

Если эти переменные есть - это точно бот!

---

## 💡 Рекомендация

**Начните с проекта `calm-exploration`** (зеленый, online) - он уже упоминался в разговоре.

1. Кликните на `calm-exploration`
2. Посмотрите список сервисов
3. Найдите сервис, который может быть ботом
4. Кликните на него
5. Проверьте переменные окружения (Settings → Variables)

---

## ✅ После того, как нашли бота

1. Кликните на сервис с ботом
2. Перейдите во вкладку **"Deployments"**
3. Проверьте последний деплой
4. Если нужно - нажмите **"Redeploy"** в Settings

---

## 🆘 Если не можете найти

Проверьте все проекты по очереди:
1. `calm-exploration` ← начните с этого
2. `protective-nourishment`
3. `remarkable-truth`
4. `airy-respect`
5. `jubilant-youth`

В каждом проекте смотрите список сервисов и ищите что-то связанное с ботом или delivery dates.
