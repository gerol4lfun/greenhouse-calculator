## 📋 ШАГ 1: Создайте новый проект

1. Откройте [Railway](https://railway.app)
2. Нажмите **"+ New"** (вверху справа)
3. Выберите **"New Project"**
4. В поле **"Project Name"** введите:
   ```
   Telegram Bot - Delivery Dates
   ```
5. Нажмите **"Create Project"**

**✅ Готово! Новый проект создан!**

**💡 Если проект создался со случайным названием (например, `serene-caring`):**
- См. инструкцию: `ПЕРЕИМЕНОВАТЬ_ПРОЕКТ.md`
- Там описано, как переименовать проект в "Telegram Bot - Delivery Dates"

---

## 📋 ШАГ 2: Подключите GitHub

1. В новом проекте нажмите **"+ New"**
2. Выберите **"GitHub Repo"**
3. Появится список ваших репозиториев
4. Найдите **`delivery-bot-telegram`**
5. Кликните на него
6. Railway спросит про папку → оставьте пустым или введите **`telegram-bot`**

**✅ Готово! Репозиторий подключен!**

## 📋 ШАГ 4: Дождитесь деплоя

1. Перейдите во вкладку **"Deployments"** (или **"Activity"**)
2. Вы увидите процесс деплоя
3. Подождите 1-2 минуты
4. Дождитесь статуса **"SUCCESS"** (зеленая галочка ✅)

**Если статус "FAILED" (красный крестик):**
- Кликните на деплой
- Посмотрите логи
- Если видите ошибку - напишите мне

---

## 📋 ШАГ 5: Проверьте логи

1. Перейдите во вкладку **"Logs"**
2. Прокрутите вниз до самых свежих логов
3. Ищите строки:
   - `✅ Supabase подключен`
   - `🤖 Бот запущен и готов к работе!`

**Если видите эти строки → бот работает! ✅**

---

## 📋 ШАГ 6: Протестируйте

1. Откройте Telegram
2. Отправьте боту: `/start`
3. Бот должен ответить!

4. Отправьте тестовое сообщение:
   ```
   Москва с 12.02, кроме 13.02, 14.02
   Питер с 9.02
   ```

5. **Ожидаемый результат:**
   - ✅ Найдено записей: **2**
   - ✅ Москва - 12.02 (кроме 13.02, 14.02)
   - ✅ Санкт-Петербург - 09.02
