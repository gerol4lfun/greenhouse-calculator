## ⚙️ Переменные окружения:

1. `TELEGRAM_BOT_TOKEN`
2. `SUPABASE_URL`
3. `SUPABASE_SERVICE_ROLE_KEY`
4. `ADMIN_USER_ID` (опционально)

---

## 📋 Как найти проект:

1. Зайдите на [railway.app](https://railway.app)
2. Ищите проект: **"Telegram Bot - Delivery Dates"**
3. Если не видите - проверьте все проекты, ищите тот, где есть сервис с GitHub репозиторием `delivery-bot-telegram`

---

## 🆘 Если не можете найти проект:

1. Откройте каждый проект по очереди
2. В каждом проекте: Settings → Variables
3. Ищите переменную `TELEGRAM_BOT_TOKEN`
4. Если нашли → это ваш проект!
