## 🔍 2. ПОИСК В FAQ (По ключевым словам)

**Проблема (v149):** Поиск работал только по точному совпадению фразы. Например, при поиске "отзывы" не находил статьи, где это слово есть в тексте.

**Исправление (v150):** Поиск теперь работает по ключевым словам - запрос разбивается на слова, и ищется каждое слово отдельно. Это позволяет находить статьи даже при частичном совпадении.

### Тест:
1. Откройте FAQ
2. Введите в поиск: "отзывы"
3. Проверьте результаты

**Ожидаемый результат:**
- [ ] Находится статья про отзывы (если она есть в базе)
- [ ] Поиск работает даже при частичном совпадении
- [ ] Можно искать несколько слов (например: "работа с клиентами")
- [ ] Если ничего не найдено, показывается сообщение "Ничего не найдено"

## 🎨 3. РАСПОЛОЖЕНИЕ КНОПКИ "ВЫЙТИ"

**Проблема (v149):** Кнопка "Выйти" была расположена криво, не в гармонии с кнопкой админ-панели.

**Исправление (v150):** Кнопка "Выйти" теперь автоматически сдвигается левее, когда админ-панель видна, чтобы они были рядом и красиво расположены.

### Тест:
1. Войдите в калькулятор
2. Проверьте расположение кнопки "Выйти"
3. Если вы админ - проверьте расположение кнопки "Админ-панель" и "Выйти" вместе

**Ожидаемый результат:**
- [ ] Кнопка "Выйти" расположена справа вверху
- [ ] Если админ-панель видна, кнопка "Выйти" сдвигается левее
- [ ] Кнопки расположены гармонично рядом друг с другом
- [ ] Кнопки не перекрываются

## 🌱 4. ВОПРОС ПРО СБОРКУ ГРЯДОК

**Проблема (v149):** При выборе грядок без сборки менеджер мог забыть включить сборку, и это не уточнялось.

**Исправление (v150):** При применении грядок без сборки появляется вопрос "Грядки выбраны. Нужна ли сборка грядок?" с возможностью сразу включить сборку.

### Тест:
1. Откройте модальное окно выбора грядок
2. Выберите любые грядки (не рекомендуемые)
3. НЕ включайте чекбокс "Сборка грядок"
4. Нажмите "Применить"

**Ожидаемый результат:**
- [ ] Появляется вопрос: "Грядки выбраны. Нужна ли сборка грядок?"
- [ ] Если нажать "Да" → сборка включается автоматически
- [ ] Если нажать "Нет" → грядки применяются без сборки
- [ ] Если сборка уже включена → вопрос не появляется

**Результат:** __________

## 🏙️ 5. СКЛОНЕНИЕ НАЗВАНИЙ ГОРОДОВ

**Проблема (v149):** Названия городов не склонялись в родительный падеж. Например, было "доставка до Истра" вместо "доставка до Истры".

**Исправление (v150):** Добавлена функция склонения названий городов. Теперь правильно: "доставка до Истры", "доставка до Москвы", "доставка до Санкт-Петербурга" и т.д.

### Тест:
1. Рассчитайте стоимость теплицы
2. Введите адрес доставки (например, в Истре, Москве, Санкт-Петербурге)
3. Проверьте короткое КП

**Ожидаемый результат:**
- [ ] "доставка до Истры" (не "до Истра")
- [ ] "доставка до Москвы" (не "до Москва")
- [ ] "доставка до Санкт-Петербурга" (не "до Санкт-Петербург")
- [ ] Другие города склоняются правильно

**Тест-кейсы для проверки:**
- Истра → Истры ✅
- Москва → Москвы ✅
- Санкт-Петербург → Санкт-Петербурга ✅
- Казань → Казани ✅
- Нижний Новгород → Нижнего Новгорода ✅
