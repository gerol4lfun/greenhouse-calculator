# 🏠 Калькулятор доставки и стоимости теплиц

## 🚀 Быстрый старт

**Откройте:** `index.html` в браузере

## 📁 Структура проекта

```
index.html          ← ОТКРЫВАТЬ ЭТОТ ФАЙЛ!
css/                ← Стили
js/                 ← Основной код калькулятора
image/              ← Изображения для FAQ
db/                 ← Миграции базы данных
sql/                ← SQL скрипты для настройки
kb/                 ← База знаний (JSON)
scripts/            ← Вспомогательные скрипты
telegram-bot/       ← Телеграм бот
docs/               ← Документация (FAQ, инструкции, контекст)
backup/             ← Резервные копии
```

## 🔐 Авторизация

Логин: `admin` (или другой из Supabase)  
Пароль: смотрите в Supabase Dashboard → Table Editor → users

## 📚 Документация

Вся документация находится в папке `docs/`:

- **FAQ:** `docs/faq/` - все файлы с FAQ
- **Инструкции:** `docs/instructions/` - инструкции по настройке и использованию
- **Контекст:** `docs/context/` - техническое задание, контекст проекта
  - **📋 ПОЛНЫЙ КОНТЕКСТ ЧАТА:** `docs/context/ПОЛНЫЙ_КОНТЕКСТ_ЧАТА.md` - **ОБЯЗАТЕЛЬНО К ПРОЧТЕНИЮ** для нового разработчика (вся история, все версии, все проблемы и решения)
- **История изменений:** `docs/CHANGELOG.md`

## 🔧 Текущая версия: v190

### Основные возможности:
- ✨ Калькулятор стоимости теплиц с доставкой
- 📋 FAQ с 6 категориями вопросов
- 🗄️ Интеграция с Supabase для авторизации и данных
- 📱 Адаптивный дизайн
- 🖼️ Галерея изображений в FAQ
- 🛏️ Система выбора грядок с рекомендуемыми вариантами
- 🎁 Динамическая система подарков (от 35к, 55к, 75к)
- 🔄 Автоматическое обновление КП при изменении подарков

## 🛠️ Разработка

```bash
# Импорт базы знаний
npm run kb:import

# Поиск в базе знаний
npm run kb:search
```
