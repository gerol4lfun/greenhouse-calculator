## 🌱 2. ВОПРОС ПРО СБОРКУ ГРЯДОК (Исправлена логика)

**Проблема (v151):** Окно с грядками не закрывалось перед вопросом про сборку. Если закрыть окно с грядками, то вопрос появлялся, но окно оставалось открытым.

**Исправление (v152-v153):** 
- Окно с грядками теперь закрывается ПЕРЕД показом вопроса про сборку
- Вопрос показывается после небольшой задержки (300мс) для плавного закрытия
- Вопрос теперь показывает стоимость сборки: "Нужна ли сборка грядок за [стоимость] рублей?"

### Тест:
1. Откройте модальное окно выбора грядок
2. Выберите любые грядки (не рекомендуемые)
3. НЕ включайте чекбокс "Сборка грядок"
4. Нажмите "Применить"

**Ожидаемый результат:**
- [ ] Окно с грядками закрывается сразу при нажатии "Применить"
- [ ] После закрытия окна (через ~300мс) появляется вопрос про сборку
- [ ] Вопрос показывает стоимость сборки: "Нужна ли сборка грядок за [стоимость] рублей?"
- [ ] Если нажать "Да" → сборка включается автоматически, показывается toast "Сборка грядок включена"
- [ ] Если нажать "Нет" → грядки применяются без сборки
- [ ] Если сборка уже включена → вопрос не появляется, окно просто закрывается
- [ ] Можно закрыть вопрос кликом на overlay (равносильно "Нет")
