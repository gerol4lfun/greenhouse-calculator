## Версия v136-v144 - ИНТЕГРАЦИЯ ГРЯДОК

### ✨ Новые возможности

1. **Система выбора грядок (грядки оцинкованные)**
   - Добавлена кнопка "Добавить грядки" в интерфейсе калькулятора
   - Модальное окно выбора грядок с двухэтапным процессом:
     - Шаг 1: Выбор типа (Низкие 19см / Высокие 38см) с изображениями
     - Шаг 2: Выбор характеристик (ширина, длина) с учетом размеров теплицы
   - Фильтрация грядок по длине теплицы (показываются только грядки не длиннее теплицы)
   - Рекомендуемый вариант грядок:
     - Для теплиц 4м: 2 боковые грядки по 1м шириной
     - Для теплиц 6м+: центральная грядка 4м + 2 боковые по длине теплицы
     - Ширина зависит от ширины теплицы (>=3.5м: 0.8м центр/1м бока, иначе: 0.5м центр/0.8м бока)
   - Два варианта добавления: "Без сборки" и "Со сборкой"
   - Автоматическое закрытие модального окна после выбора рекомендуемого варианта
   - Автоматический пересчет стоимости при добавлении грядок

2. **Цены на грядки**
   - Данные хранятся локально в массиве `BEDS_DATA` (40 вариантов: низкие/высокие × 4 ширины × 5 длин)
   - Цены одинаковые для всех городов (захардкожены в коде)
   - Ширины: 0.5м, 0.65м, 0.8м, 1м
   - Длины: 4м, 6м, 8м, 10м, 12м
   - Высоты: 19см (низкие), 38см (высокие)

3. **Сборка грядок**
   - Опциональная услуга с ценами в зависимости от длины теплицы:
     - 4м: 990 руб
     - 6м: 1490 руб
     - 8м: 1990 руб
     - 10м: 2490 руб
     - 12м: 2990 руб
     - 14м: 3490 руб
     - 16м: 3990 руб
   - Чекбокс "Добавить сборку грядок" в модальном окне
   - Динамический расчет стоимости сборки (за 1 грядку или за все грядки)

4. **Интеграция в коммерческое предложение**
   - Грядки добавляются в длинное КП: "Грядки оцинкованные: [тип] [размеры] - [цена] руб."
   - Сборка грядок: "Сборка грядок (за 1 грядку) - XXXX руб." или "Сборка грядок (за все грядки) - XXXX руб."
   - В коротком КП: "Грядки: XXXX рублей (со сборкой)" или "Грядки: XXXX рублей"
   - Грядки учитываются в общей стоимости теплицы

5. **Система уведомлений (Toast)**
   - Замена стандартных `alert()` на стильные toast-уведомления
   - Типы: success, error, warning, info
   - Автоматическое исчезновение через 4-5 секунд
   - Неблокирующие уведомления

### 🔧 Технические изменения

- **Новые функции в `js/scripts.js`:**
  - `showBedsModal()` - открытие модального окна грядок
  - `closeBedsModal()` - закрытие модального окна
  - `renderBedsSelection()` - рендеринг выбора грядок
  - `renderBedTypeSelection()` - выбор типа грядок (низкие/высокие)
  - `renderBedCharacteristics()` - выбор характеристик грядок
  - `getRecommendedBeds(height)` - расчет рекомендуемого варианта
  - `applyRecommendedBeds(withAssembly)` - применение рекомендуемого варианта
  - `changeBedQuantity(bedId, delta)` - изменение количества грядок
  - `updateBedsCounter()` - обновление счетчика на кнопке
  - `calculateBedsAssemblyCost(selectedBeds)` - расчет стоимости сборки
  - `updateBedsAssemblyPrice()` - обновление цены сборки в модальном окне
  - `getGreenhouseDimensions()` - получение размеров теплицы
  - `showToast()`, `showSuccess()`, `showError()`, `showWarning()`, `showInfo()` - система уведомлений

- **Новые константы:**
  - `BEDS_DATA` - массив с данными о грядках (40 элементов)
  - `BEDS_ASSEMBLY_PRICES` - цены на сборку по длине теплицы

- **Изменения в существующих функциях:**
  - `calculateGreenhouseCost()` - добавлен расчет стоимости грядок и сборки
  - `generateCommercialOffer()` - добавлен вывод грядок в длинное КП
  - `generateShortOffer()` - добавлен вывод грядок в короткое КП

- **Новые элементы в `index.html`:**
  - Кнопка "Добавить грядки" с бейджем счетчика
  - Модальное окно выбора грядок
  - Контейнер для toast-уведомлений

- **Новые стили в `css/styles.css`:**
  - Стили для кнопки грядок, модального окна, карточек выбора
  - Стили для рекомендуемого варианта
  - Стили для toast-уведомлений

### 🐛 Исправленные ошибки

- Исправлены дубликаты объявления переменных (`assemblyCheckbox` → `bedsAssemblyCheckbox`)
- Исправлена функция `applyRecommendedBeds` (добавлен параметр `withAssembly`, правильная логика)
