# 📚 Документация проекта

## 📁 Структура документации

### FAQ (`faq/`)
Все файлы с вопросами и ответами для FAQ:
- `FAQ_ПОЛНЫЙ_СПИСОК_ДЛЯ_РЕДАКТИРОВАНИЯ.md` - полный список FAQ
- `FAQ_СТРУКТУРА_ДЛЯ_ВНЕДРЕНИЯ.md` - структура для внедрения
- `FAQ_ОТФОРМАТИРОВАНО_*.md` - отформатированные разделы FAQ

### Инструкции (`instructions/`)
- `ИНСТРУКЦИЯ_ПАРОЛЬ.md` - инструкции по паролям
- `ИСПРАВЛЕНИЕ_СЕРВЕРА_CURSOR.md` - решение проблем с Cursor
- `GITHUB_PAGES_SETUP.md` - настройка GitHub Pages
- `OPTIMIZATION_REPORT.md` - отчет об оптимизации
- `TELEGRAM_BOT_PLAN.md` - план телеграм бота

### Контекст (`context/`)
- `ТЗ.md` - техническое задание
- `КОНТЕКСТ_ПРОЕКТА.md` - контекст проекта
- `КОНТЕКСТ_ЧАТА_FAQ_SAFARI.md` - контекст чата FAQ Safari
- `доска.md` - контент для FAQ
- `АНАЛИЗ_ТЕЛЕГРАМ_ПОСТОВ_ДЛЯ_FAQ.md` - анализ постов

### Архив (`archive/`)
Старые и неактуальные файлы:
- `FAQ_ПОЛНЫЙ_СПИСОК_ДЛЯ_РЕДАКТИРОВАНИЯ copy.md` - дубликат

### Общие файлы
- `CHANGELOG.md` - история изменений проекта
