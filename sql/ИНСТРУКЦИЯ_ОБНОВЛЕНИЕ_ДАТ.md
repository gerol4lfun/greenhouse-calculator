# Как обновить даты доставки вручную

## Шаг 1: Открыть Supabase

1. Зайдите на https://supabase.com
2. Откройте свой проект
3. Слева выберите **SQL Editor**

## Шаг 2: Выполнить скрипт

1. Нажмите **New query** (или откройте новый запрос)
2. Откройте файл `UPDATE_ALL_DATES_FROM_USER_LIST.sql` из папки `sql/`
3. Скопируйте **весь** текст из файла
4. Вставьте в окно SQL Editor
5. Нажмите **Run** (зелёная кнопка)

Должно появиться сообщение: **Success. No rows returned** или подобное.

## Шаг 3: Проверить в калькуляторе

1. Откройте калькулятор (index.html в браузере)
2. Нажмите кнопку **Даты доставки**
3. Убедитесь, что даты соответствуют введённым в скрипт
4. Нажмите **Ctrl+Shift+R** (или Cmd+Shift+R на Mac), чтобы обновить страницу без кеша

---

**Если нужно изменить даты** — отредактируйте файл `UPDATE_ALL_DATES_FROM_USER_LIST.sql` (замените даты на нужные) и выполните его снова в Supabase.
