# SQL-скрипты для Supabase

## Даты доставки (основное)

| Файл | Назначение |
|------|------------|
| `FIX_DELIVERY_DATES_CLEANUP.sql` | **Выполнить один раз.** Очистка дубликатов «Город доставки», добавление 32 городов, RPC для обновления |
| `UPDATE_ALL_DATES_FROM_USER_LIST.sql` | Ручное массовое обновление дат — отредактировать значения и выполнить в SQL Editor |
| `ИНСТРУКЦИЯ_ОБНОВЛЕНИЕ_ДАТ.md` | Краткая инструкция по ручному обновлению |

Обычно даты обновляются через Telegram-бота. Ручные скрипты — на случай сбоев.

## Папка archive/

Старые и разовые скрипты (диагностика, слияние дубликатов и т.п.). Использовать только при необходимости.

## База знаний и грядки

- База знаний: см. `db/migrations/` и `sql/archive/README_KNOWLEDGE_BASE.md`
- Грядки: инструкции в `sql/archive/` (если нужны)
