# 🚀 Готовые команды для загрузки на GitHub

**Просто копируйте и вставляйте команды по порядку в Терминал!**

---

## ✅ КОМАНДА 1: Настройка Git

Скопируйте и выполните:

```bash
git config user.name "gerol4lfun"
```

Нажмите Enter.

---

## ✅ КОМАНДА 2: Настройка email

Скопируйте и выполните:

```bash
git config user.email "figment-nucleus-2b@icloud.com"
```

Нажмите Enter.

---

## ✅ КОМАНДА 3: Добавление всех файлов

Скопируйте и выполните:

```bash
git add .
```

Нажмите Enter. (Ничего не произойдет визуально - это нормально!)

---

## ✅ КОМАНДА 4: Проверка (опционально)

Скопируйте и выполните:

```bash
git status
```

Нажмите Enter. Вы должны увидеть список файлов зеленым цветом.

---

## ✅ КОМАНДА 5: Создание коммита

Скопируйте и выполните:

```bash
git commit -m "Initial commit: калькулятор теплиц v147"
```

Нажмите Enter. Вы увидите сообщение о создании коммита.

---

## ✅ КОМАНДА 6: Подключение к GitHub

Скопируйте и выполните:

```bash
git remote add origin https://github.com/gerol4lfun/greenhouse-calculator.git
```

Нажмите Enter. (Ничего не произойдет - это нормально!)

---

## ✅ КОМАНДА 7: Отправка файлов на GitHub

Скопируйте и выполните:

```bash
git push -u origin main
```

Нажмите Enter.

**Если попросит авторизацию:**
- **Username:** `gerol4lfun`
- **Password:** используйте **Personal Access Token** (не обычный пароль!)

**Если токена нет:**
1. Откройте: https://github.com/settings/tokens
2. Нажмите "Generate new token" → "Generate new token (classic)"
3. Название: `greenhouse-calculator`
4. Срок: выберите (например, 90 дней)
5. Отметьте галочку **`repo`** (полный доступ к репозиториям)
6. Нажмите "Generate token" внизу
7. **Скопируйте токен** (он показывается только один раз!)
8. Используйте этот токен как пароль

---

## ✅ Готово!

После выполнения всех команд откройте:
https://github.com/gerol4lfun/greenhouse-calculator

Вы должны увидеть все файлы проекта!
